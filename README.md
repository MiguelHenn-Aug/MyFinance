## MyFinance

Com base em algumas necessidades, decidi criar este protótipo de aplicação web focada em controle de gastos, o objetivo aqui é muito simples:

- O usuário traz o valor que deseja com o qual desejar iniciar no sistema;
- Pode adicionar RECEBIMENTOS e GASTOS para os cálculos;
- Indica as datas para as respectivas transações;
- Pode visualizar, em tempo real, a dashboard com todos os dados requisitados.

A linguagem que escolhi para esse pequeno projeto foi PHP, com os frameworks Laravel (para backend) e Blade (para frontend). Escolhi essa linguagem e frameworks por fins acadêmicos, atualmente eles vêm sendo o principal cerne dos meus estudos então achei justo que escolhesse-os para esse projeto.

## Requisitos Funcionais

Foi planejado que esse sistema fosse intuitivo para o usuário, sendo assim, suas funções já foram bem definidas:
- Criação de Saldo base;
- Adição de valores com o Recebimento;
- Subtração de valores com Gastos;
- Saldo futuro como resultado das operações;
- Saldo base se mantém inalterado, até que o usuário altere-o ou a data de um gasto/recebimento chegue;
- Gráfico baseado em valores recebidos e gastos por data.

Caso decida fazer uso ou testar este protótipo, adoraria receber um feedback, obrigado pela atenção!
